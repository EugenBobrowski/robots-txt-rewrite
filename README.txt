Manage your robots.txt form admin side

== Description ==

A simple plugin to manage your robots.txt WordPress returning.

Plugin provide to discourage search engines from indexing site. Use the stock WordPress option.


== Installation ==

1. Upload `robots-txt-rewriter.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings / Robots.txt Options and just manage.
